# 🎯 Трекер целей

> Простой способ дойти до цели — шаг за шагом, без давления дедлайнов и лишней сложности.

**[Открыть приложение →](https://bobah1988.github.io/goal-tracker/)**

## Идея

Я перепробовал немало сервисов и методик постановки целей — и почти все страдали одним и тем же: даты, которые давят дедлайном, обязательные формулировки по SMART, графики, картинки и прочий шум, который только отвлекает от самого главного — движения к цели.

Трекер целей устроен иначе. Никаких обязательных сроков и сложных рамок — только цель и путь к ней.

Логика предельно простая:

1. Пишете цель.
2. Если она кажется большой — делите на подцели.
3. Если подцель всё ещё кажется сложной — делите и её.
4. Так, маленькими шагами, вы видите, как закрываете задачу за задачей и приближаетесь к своей мечте.

Я делал этот проект в первую очередь для себя — и решил открыть его всем, кому такой подход к целям окажется близким и полезным.

## Возможности

- **Дерево целей** — цель делится на подцели любой вложенности, прогресс считается автоматически от выполненных шагов.
- **Чек-листы** — к любому шагу можно прикрепить список мелких пунктов; шкала выполнения растёт по мере того, как вы их вычёркиваете.
- **Колесо баланса** — наглядный обзор всех сфер жизни на одном экране.
- **Без дедлайнов и рамок** — никаких обязательных дат и методик, только цель и шаги к ней.
- **Работает офлайн** — устанавливается как приложение (PWA) на телефон или компьютер.
- **Личный аккаунт (по желанию)** — вход по email синхронизирует прогресс между устройствами; без аккаунта данные хранятся локально в браузере.
- **Русский и English** — переключение языка в один клик.

## Технологии

Один статичный файл на чистых HTML/CSS/JavaScript — без сборки и фреймворков. Хостится на GitHub Pages. Синхронизация между устройствами — Firebase (Authentication + Firestore), полностью опциональна: без неё приложение работает так же, но данные остаются только в этом браузере.
